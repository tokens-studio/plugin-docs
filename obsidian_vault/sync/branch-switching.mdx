# Branch Switching

Now you can view and switch branches of your remote sync.

Click the `branch` icon at the bottom of the plugin to see the list of branches.
 
 
<img src="/branch-switch.png" alt="Switching branches in Tokens Studio for Figma" width="500"/>
 
Click on the branch name to view all the branches and select a branch you want to switch to.
 
 
<img src="/branch-switch-1.png" alt="Switching branches in Tokens Studio for Figma" width="500"/>
 
 
You can also create a new branch by clicking on `Create new branch from` and selecting the branch you want to duplicate.
 
 
<img src="/branch-switch-create-new.png" alt="Create new branch from existing branch" width="500"/>
